# Infrastructure & NOC Home Lab

## Objective
This project documents my hands-on infrastructure lab built using VirtualBox to strengthen Linux and Windows troubleshooting skills aligned with Network Operations roles.

## Lab Environment
- Ubuntu Server (LTS)
- Windows Server (Evaluation)
- VirtualBox
- NAT Networking

## Skills Practiced
- System monitoring & health validation
- Service troubleshooting
- Log analysis
- Patch management simulation
- Incident response & documentation
- Maintenance validation & switchover testing

This lab simulates real-world NOC scenarios in a controlled environment.
